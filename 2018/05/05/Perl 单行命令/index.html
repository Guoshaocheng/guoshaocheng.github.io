Perl单行命令

Posted on 2018-05-05 | In Perl

命令行参数

-p

  • 自动循环加自动输出，相当于 while (<>) {script; print; }

  • -p 开关更适用于对文件进行的完全处理

-n

  • 自动循环，相当于 while(<>) {script; print; }
  • -n 开关更适用于 选择性文件处理,这样的处理只需打印特定数据

-l

  • 对输入内容自动 chomp，对输出内容自动添加换行

-i <bak>

  • 原地替换文件，并将旧文件用指定后缀扩展名备份
  • 不指定扩展名则不备份

-e

  • 执行制定的脚本

-a

  • 自动分割模式，用空格分隔 $_ 并保存到 @F 中
  • 相当于 @F = split ' '，分隔符可以使用 -F 参数指定

-F

  • 指定 -a 的分隔符，可以使用正则表达式

  • -0 <num> (用 8 进制表示)

    • 指定记录分隔符，默认为换行
  • -00
    • 段落模式，即使用连续换行作为分隔符
  • -0777
    • 禁用分隔符，即将所有文件作为一个记录
